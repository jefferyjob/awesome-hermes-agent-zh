# Contributor Covenant 行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让每个人都能在我们的社区中获得免受骚扰的参与体验，无论其年龄、体型、可见或不可见的残障、族裔、性特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教，或性身份与性取向如何。

我们承诺以有助于建设开放、友好、多元、包容且健康社区的方式行动与互动。

## 我们的标准

有助于为社区营造积极环境的行为包括：

* 对他人展现同理心与善意
* 尊重不同的意见、观点和经历
* 提供建设性反馈，并以得体的方式接受建设性反馈
* 对自己的错误承担责任，向受到影响的人致歉，并从经历中学习
* 关注的不仅是个人利益，也包括整个社区的整体福祉

不可接受的行为包括：

* 使用带有性意味的语言或图像，以及任何形式的性关注或性挑逗
* 挑衅、侮辱性或贬损性评论，以及针对个人或政治立场的攻击
* 公开或私下骚扰
* 未经明确许可，发布他人的私人信息，例如住址或电子邮箱地址
* 其他在专业环境中可被合理视为不当的行为

## 执行

任何辱骂、骚扰或其他不可接受行为的事件，都可以报告给项目维护者。所有投诉都会被及时且公正地审查与调查。
